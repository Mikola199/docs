# Спецификация карточки вакансии

## Ключевые элементы

| Элемент | Детали | UX notes |
| --- | --- | --- |
| Название позиции | Крупный шрифт, выделено | Быстро понять суть |
| Компания | Логотип + текст | Минимизировать текст |
| Локация | Город + удаленно | Карта по клику |
| Заработок | Диапазон | Цвет выделения для high pay |
| Тип занятости | Полная / частичная / вахта | Значки для визуала |
| Кнопка отклика | CTA | Доступно всегда при скролле |

## Рекомендации по приоритету в интерфейсе

1. В верхней части карточки размещать название позиции и зарплату как основные триггеры клика.
2. Для блока компании использовать компактный логотип и короткое название (без юридических форм).
3. Для локации показывать чипы: `Город` и `Удаленно`, с возможностью открыть карту по клику.
4. Для зарплаты добавить цветовой акцент и бейдж `High pay`, если верхняя граница диапазона выше порога.
5. Для типа занятости использовать иконки рядом с текстом для быстрого сканирования списка.
6. Кнопку отклика закреплять (sticky) при скролле, чтобы не терять конверсию.
